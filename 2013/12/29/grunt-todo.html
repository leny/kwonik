Retrouvez les TODO dans vos codes avec grunt-todo

Lors de mes développements, j’ai pris l’habitude, comme beaucoup d’autres développeurs, de baliser mon code avec des TODO, des FIXME et des NOTE…

Ces balises permettent de retrouver facilement des portions de code particulières, à suivre au long du processus.

Toutefois, il faut encore pouvoir retrouver facilement toutes ces balises : jusqu’à peu, j’utilisais un plugin nommé sublime-todos, pour le fantastique éditeur Sublime Text.
Malheureusement, avec la mise à jour de Sublime Text en v3, le plugin ne marche plus.

J’ai donc décidé de créer mon propre plugin.
Ce n’est pas un plugin pour Sublime Text, mais pour Grunt, le fantastique taskrunner en node.js

Je vous invite à découvrir ce petit plugin, nommé grunt-todo, et lire sa documentation pour apprendre à l’utiliser, c’est simple et pratique.
