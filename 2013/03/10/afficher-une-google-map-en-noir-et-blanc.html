Afficher une google map en noir & blanc

Un p’tit truc vite fait (pour tester le code dans les billets de blog), comment afficher une google map en noir & blanc sur son site.

L’API des GoogleMaps est une api vraiment très sympa et qui mérite qu’on s’y attarde : simple à prendre en main mais terriblement puissante.

En admettant que vous avez inclus les scripts nécéssaires et que vous avez une div dont l’id est gmap dans votre document html, voici comment faire pour générer une carte routière de Belgique en noir & blanc, centrée sur Bruxelles.

C’est tout beau, c’est tout simple, c’est magique.
